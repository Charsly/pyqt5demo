# pyqt5demo

```
非常方便好用的左界面 + 右控制台形式。

python代码中任意print 控制台日志，自动显示在右边控制台中，方便知道点击按钮背后实时发生了什么。

因为做了sys.std 重定向，不需要大幅修改现有代码来实际操作控制台控件,就能达到代码中的任意print和控制台日志自动显示在控制台的效果。

这个demo可以作为左界面右控制台布局的万能通用pyqt5基类。

```
